# Pig Game

A classic dice game built while following Jonas Schmedtmann's "The Complete JavaScript Course".

## 🎮 About the Project

This is my first complete JavaScript project. Players take turns rolling a dice and can choose to "Hold" their current score or continue rolling. If a player rolls a 1, they lose their current points and the turn passes to the other player. First to reach 100 points wins!

## ✨ Features

- Turn-based gameplay with two players
- Roll dice and accumulate points
- Hold current score to bank points
- Win condition (first to 100 points)
- Clean UI with active player highlighting
- Reset game functionality

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (DOM manipulation, event handling, game logic, sound)

## 🚀 How to Play

1. Click **Roll Dice** to roll
2. Click **Hold** to save your current points
3. First player to reach **100 points** wins!
4. Rolling a **1** loses your current turn points
